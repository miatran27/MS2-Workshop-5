Where to put your audio files
=============================

Drop your MP3 files into this folder: /audio

Suggested filenames (used by the app):
  - bedstuy.mp3       (Bedford-Stuyvesant, Brooklyn)
  - west-village.mp3  (West Village, Manhattan)
  - union-square.mp3  (Union Square, Manhattan)
  - central-park.mp3  (Central Park, Manhattan)

If a file is missing, the popup will show: "Audio not found — add the file to /audio"
